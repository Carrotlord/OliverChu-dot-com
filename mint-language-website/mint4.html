 Mint 4.0
Speed optimizations and new features

Mint 2.9 interprets abstract syntax trees at runtime to execute programs. Mint 4.0, currently under development, compiles source code into bytecode and executes it on a virtual machine.

After a few speed tests, Mint 4.0 handles data and calculations much faster than Mint 2.9, and has a smaller memory footprint. New language features will be added in addition to these optimizations. Stay tuned for more updates!
